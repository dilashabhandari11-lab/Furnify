# Furnify

Furnify is a modern furniture e-commerce web application built using React. It allows users to browse a variety of furniture items, including both new and thrift (second-hand) products through a clean and responsive interface.

## Features
- Browse furniture products
- Separate categories for New and Thrift items
- Responsive and user-friendly UI
- Product cards with image, price, and details

## Tech Stack
- React.js
- HTML5
- CSS3
- JavaScript

## Getting Started

1. Clone the repository:
   git clone https://github.com/dilashabhandari11/furnify.git

2. Navigate to the project folder:
   cd furnify

3. Install dependencies:
   npm install

4. Run the app:
   npm start

## Purpose
This project was developed to improve frontend development skills using React, focusing on component-based architecture and UI design.
